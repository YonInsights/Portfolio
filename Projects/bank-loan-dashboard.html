Bank Loan Report Dashboard

Project Overview

This project involves designing a dynamic and interactive Excel dashboard to analyze and visualize bank loan data for a client in the financial domain. The dashboard provides insights into key performance indicators (KPIs) such as total loan applications, funded amounts, repayment status, average interest rates, and depth-to-income (DTI) ratios.

The report categorizes loans into "Good Loans" (Fully Paid or Current) and "Bad Loans" (Charged Off), enabling risk assessment and decision-making.

Key Features

Dynamic KPI dashboard with real-time updates and automated calculations
Interactive data filtering with slicers and drill-down capabilities
Advanced visualizations including pie charts, line charts, and map charts
Risk assessment module with loan categorization and analysis
Custom reporting templates and export functionality
Data cleaning and validation with error handling

Tools & Technologies

  • Microsoft Excel
  • Pivot Tables & Charts
  • Advanced Filters
  • Excel Functions
  • Data Visualization
  • SQL Integration
