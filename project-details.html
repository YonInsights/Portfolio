Project Description

This project involves designing a dynamic and interactive Excel dashboard to analyze and visualize bank loan data for a client in the financial domain. The dashboard provides insights into key performance indicators (KPIs) such as total loan applications, funded amounts, repayment status, average interest rates, and depth-to-income (DTI) ratios.

The report categorizes loans into "Good Loans" (Fully Paid or Current) and "Bad Loans" (Charged Off), enabling risk assessment and decision-making. It also includes visualizations like pie charts, line charts, map charts, and slicers to allow users to filter and explore data by various parameters such as month, state, loan purpose, and employee length.

The goal is to create a professional, visually appealing, and functional Excel dashboard that mirrors previous reports made in Power BI and Tableau, ensuring consistency across platforms.

Project Requirements

Data Source:

  • Raw loan data in CSV format
  • Optional SQL Server integration for direct data import

Dashboard Features:

  • Dynamic KPIs showing:
    • Total loan applications
    • Total funded amount
    • Total received payments
    • Average interest rate
    • DTI ratio
    • Good vs. Bad loan percentages
  • Visual charts including:
    • Pie/Donut charts (loan distribution)
    • Line charts (monthly trends)
    • Map charts (state-wise loan distribution)
    • Bar charts (employee length analysis)
    • Tree maps (home ownership analysis)

Tools Used in the Project

  • Pivot Tables & Charts
  • Slicers
  • Advanced Formulas
  • Formatting Tools
  • SQL Server
